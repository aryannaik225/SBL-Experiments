Experiment 4 – Create a Declarative Jenkins Pipeline
Steps
1. new item :
    enter an item name : MyScriptedPipeline
    select pipeline
    
2. select pipeline in configure
CODE--------------------------------------------------------------------
pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                echo 'Building...'
            }
        }
        stage('Test') {
            steps {
                echo 'Testing...'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Deploying...'
            }
        }
    }
}


3.Create New Pipeline Job in Jenkins

-Dashboard → New Item → name: DemoPipeline

-Select Pipeline → OK

-Scroll to Pipeline section → choose Pipeline script from SCM

-SCM = Git → enter your repo URL (and credentials if private)

4.Run Pipeline
-Click Build Now

-View results in Stage View
